# EasyPS - 快捷签章

一款专注于电子合同签署场景的iOS原生应用。

## 项目简介

快捷签章（QuickStamp & Sign）致力于解决电子合同签署中的核心痛点：
- 📸 智能印章处理：将纸质印章转换为透明背景的电子印章
- ✍️ 手写签字：流畅的签名体验，支持 Apple Pencil
- 📄 PDF文档处理：在文档中添加印章和签名
